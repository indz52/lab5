Автоматическая документация кода (Модули)

class lvl_system.LvlSystem(hp, gold)[исходный код]

Базовые классы: object

Класс для работы с системой уровней (LVL) главного героя.

Отвечает за накопление опыта, повышение уровня и распределение бонусов.

Примечание

Текущая реализация использует приватные атрибуты для защиты данных игрока от прямой модификации извне.

Параметры:

  • hp (int or float) – Начальное количество очков здоровья персонажа.

  • gold (int) – Начальное количество золота у персонажа.

Пример использования:

hero_lvl = LvlSystem(hp=100, gold=50)
hero_lvl.add_xp(350)  # Автоматически повысит уровень
__lvl_up_points_amount = 300

Количество очков опыта, необходимое для базового повышения уровня. Расчет итогового порога: \(current\_lvl \times lvl\_up\_points\_amount\)

add_xp(amount)[исходный код]

Добавляет персонажу очки опыта (XP).

После добавления автоматически запускается проверка на повышение уровня lvl_up().

Параметры:

amount (int) – Количество добавляемого опыта.

План

Реализовать валидацию входящего значения (amount не должно быть отрицательным).

lvl_up()[исходный код]

Проверяет условия и осуществляет повышение уровня героя.

Если очков опыта достаточно, списывает их, увеличивает уровень на 1 и вызывает метод распределения наград lvl_up_bonus().

lvl_up_bonus()[исходный код]

Интерактивный метод выбора бонуса при достижении нового уровня.

Предоставляет игроку выбор в консоли:

  1. Получить 100 золота

  2. Увеличить максимальное здоровье на 20%

Предупреждение

Метод использует блокирующий ввод input(), что может остановить игровой цикл в реальной игре.

show_lvl()[исходный код]

Выводит в консоль информацию о текущем уровне персонажа.

См.также

О методе повышения уровня см. в lvl_up().
