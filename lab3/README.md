# Документация проекта: Игровая система уровней (LvlSystem)

---

## 📌 Общая информация
* **Автор:** Павлов Александр
* **Язык разработки:** Python 3.x
* **Инструмент сборки:** Sphinx (с поддержкой Markdown/reST)

Этот программный модуль предназначен для автоматизации механики прокачки персонажей в RPG-играх. Он берет на себя расчет опыта, отслеживание текущего уровня и выдачу внутриигровых наград.

---

## 🛠️ Основной функционал (Модули)
Документация разбита на логические блоки:
1. **[Модуль Уровней (lvl_system)](#)** — содержит описание основного класса `LvlSystem`.
2. **Вспомогательные блоки** — методы расчета бонусов и логирования прогресса.
